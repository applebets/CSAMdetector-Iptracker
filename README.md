# Warning:
Not all sites behave the same. Some, like LimeTorrents, include features that make scraping significantly more difficult. However, most of the magnet sites we plan to target do not use these protections, so we should be fine.

Features
Metadata scanning with Jakket API
Keyword flagging from predefined lists
NSFW detection on images/videos (frame slicing for video)
IP tracing using IPWhois API
Proxy/VPN detection
Ethical Safeguards
Tested only on legal NSFW datasets
Never automated or deployed publicly
Intended as a conceptual framework for use only under supervision of authorized bodies

Future Directions
Collaborations with law enforcement
Improved NSFW classifiers with deep learning
Automated reporting pipelines
